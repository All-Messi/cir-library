<p class=MsoNormal style='margin-bottom:0in;line-height:normal'><span
style='font-size:12.0pt;font-family:"Times New Roman",serif'>The windows
keyboard uses Unicode, which is a true Greek font. There is a code for every
character, including for example, alpha smooth breathing and circumflex. There
are some older programs that were easy to use, but the computer didn't really
see them as Greek letters. They saw them as [a / ^] or something like that.
Unicode is best. It is always Greek. You can use it online, in emails,
whatever. You can alphabetize and search texts.</span></p>
